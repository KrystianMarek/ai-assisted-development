# Technical Guidance Documents

This folder contains requests for expert guidance and the responses received.

## Naming Convention

```
NNN-request-<topic>.md   — Outgoing request for guidance (drafted by agent or human)
NNN-response-<topic>.md  — Received guidance/response (saved by the user only)
```

> **Rule:** AI agents may draft **request** documents. Agents must NOT generate **response** documents — those are authored by the external expert and saved by the user.

## How to Request Guidance

1. Create a new request file: `NNN-request-<topic>.md` (next sequential number).
2. Include:
   - Context and current decisions
   - Specific questions
   - Code examples if relevant
   - Priority order
3. Share with the consultant/expert (external to this project).
4. **User** saves the response as `NNN-response-<topic>.md`.
5. Update this README index.

## Request Template

```markdown
# Guidance Request: <Topic>

## Context
Current situation and background.

## Current Approach
What we're doing or planning.

## Questions
1. Specific question 1
2. Specific question 2

## Constraints
- Budget, timeline, team expertise limitations

## Priority
Which questions are most critical?
```
